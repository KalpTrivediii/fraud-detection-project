## 🚀 Overview
This project focuses on detecting fraudulent transactions using **machine learning and anomaly detection techniques**.  
It simulates a real-world fraud detection pipeline used in fintech systems.

---

## ⚙️ Features

✨ **Anomaly Detection**
- Uses Isolation Forest to identify unusual transactions

📊 **Risk Identification**
- Flags high-risk transactions based on patterns

🔍 **Data Analysis**
- Processes transaction data for insights and detection
